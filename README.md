# AI-Powered-Diagnostic-Tool-for-Robust-Brain-Tumor-Detection-and-Multi-Class-Classification

This project aims to develop an AI-powered diagnostic tool that automatically detects and classifies brain tumours from MRI scans using deep learning algorithms. The workflow begins by accessing and preprocessing high-quality datasets of MRI scans from various sources. Next, the model will be trained to detect the presence of a tumour, segment it to identify tissues, and extend it for multi-class classification of different tumour types. The final product will be a desktop application that inputs MRI scans and outputs a diagnosis. By automating the analysis of MRI data, the tool will act as clinical decision-support to aid radiologists in making faster and more reliable diagnoses.
